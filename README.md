# Sass RWD Mock Exam – CodersLab

This project was created as part of the **CodersLab – JavaScript Frontend Developer** course.

## Project overview
The goal of this project was to practice **Sass (SCSS)**, **responsive web design**, and **mobile-first approach** by completing a set of structured tasks during a workshop / mock exam.

## What was implemented
- Custom CSS reset using SCSS
- Color system based on Sass maps and loops
- Reusable mixins (e.g. arrow generator)
- Custom 10-column grid system built with Sass
- Responsive layouts using **Flexbox**
- Media queries following **mobile-first** principles
- Image responsiveness with preserved aspect ratio

## Technologies
- HTML5
- SCSS (Sass)
- CSS Flexbox
- Responsive Web Design (RWD)

## Notes
This repository represents a **course assignment** and serves as a learning and practice project.
